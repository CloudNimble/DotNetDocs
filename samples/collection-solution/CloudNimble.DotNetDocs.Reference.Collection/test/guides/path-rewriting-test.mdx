## Code Block Preservation Test

Paths inside code blocks should NOT be rewritten:

```javascript
// This import path should remain unchanged
import { Component } from '/snippets/Component.jsx';

// This image path should remain unchanged
const logo = '/images/logo.png';
```

```bash
# These paths should also remain unchanged
curl https://api.example.com/images/test.png
cat /snippets/example.jsx
```

## Summary

If this page renders correctly in a collection:
- Imports should work (component above should render)
- Images should display
- Links should navigate correctly (to `/test/...` paths)
- Code blocks should show original paths unchanged
