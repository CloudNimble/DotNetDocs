## ‚ö†Ô∏è Special Character Handling

<Warning>
  Special characters in XML documentation must be properly escaped:
  - `<` becomes `&lt;`
  - `>` becomes `&gt;`
  - `&` becomes `&amp;`
  - `"` becomes `&quot;`
  - `'` becomes `&apos;`
</Warning>

<Info>
  The documentation renderer handles these escapes automatically when generating output formats.
</Info>
