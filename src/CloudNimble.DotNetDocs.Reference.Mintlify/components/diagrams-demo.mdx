# Mermaid Diagrams Demo

<Note>
  Mermaid diagrams provide visual representations of inheritance hierarchies, relationships, workflows, and architecture.
</Note>

## üèóÔ∏è Class Inheritance Diagram

```mermaid
classDiagram
    class IDisposable {
        <<interface>>
        +Dispose() void
    }
    
    class IAsyncDisposable {
        <<interface>>
        +DisposeAsync() ValueTask
    }
    
    class BaseService {
        <<abstract>>
        #logger ILogger
        #configuration IConfiguration
        +Initialize() void
        +ProcessAsync()* Task
    }
    
    class DataService {
        -repository IRepository
        +GetDataAsync(id) Task~T~
        +SaveDataAsync(data) Task
        +ProcessAsync() Task
    }
    
    class CachedDataService {
        -cache IMemoryCache
        +GetDataAsync(id) Task~T~
        +InvalidateCache() void
    }
    
    IDisposable <|.. BaseService : implements
    IAsyncDisposable <|.. BaseService : implements
    BaseService <|-- DataService : inherits
    DataService <|-- CachedDataService : inherits
    
    style IDisposable fill:#ddd6fe,stroke:#7c3aed
    style IAsyncDisposable fill:#ddd6fe,stroke:#7c3aed
    style BaseService fill:#bfdbfe,stroke:#3b82f6
    style DataService fill:#bbf7d0,stroke:#10b981
    style CachedDataService fill:#fed7aa,stroke:#f97316
```

## üîÑ State Machine Diagram

```mermaid
stateDiagram-v2
    [*] --> Idle
    Idle --> Initializing : Start()
    Initializing --> Ready : InitComplete
    Initializing --> Failed : InitError
    
    Ready --> Processing : Process()
    Processing --> Ready : Success
    Processing --> Failed : Error
    Processing --> Processing : Retry
    
    Ready --> Stopping : Stop()
    Failed --> Stopping : Stop()
    Processing --> Stopping : Cancel()
    
    Stopping --> Stopped : CleanupComplete
    Stopped --> [*]
    
    Failed --> Initializing : Reset()
```

## üåä Flowchart

```mermaid
flowchart TD
    Start([Start]) --> Input[/Receive Request/]
    Input --> Validate{Valid?}
    
    Validate -->|No| Error[Return Error]
    Error --> End([End])
    
    Validate -->|Yes| CheckCache{In Cache?}
    CheckCache -->|Yes| ReturnCache[Return Cached]
    ReturnCache --> End
    
    CheckCache -->|No| Process[Process Request]
    Process --> CheckDB{Exists in DB?}
    
    CheckDB -->|Yes| Update[Update Record]
    CheckDB -->|No| Create[Create Record]
    
    Update --> SaveCache[Save to Cache]
    Create --> SaveCache
    
    SaveCache --> Success[Return Success]
    Success --> End
```
