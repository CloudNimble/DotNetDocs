# Quick Start Guide

Get up and running with DotNetDocs documentation in just a few steps.

<Steps>
  <Step title="Install the Package">
    Install the DotNetDocs package from NuGet:
    
    <CodeGroup>
      ```bash .NET CLI
      dotnet add packag eDotNetDocs.Tools
      ```
      
      ```xml PackageReference
      <PackageReference Include="DotNetDocs.Core" Version="1.*-*" />
      ```
      
      ```powershell Package Manager
      Install-Package DotNetDocs.Core
      ```
    </CodeGroup>
  </Step>
  
  <Step title="Add Using Statements">
    Import the namespaces you need:
    
    ```csharp
    using CloudNimble.DotNetDocs.Tests.Shared;
    using CloudNimble.DotNetDocs.Tests.Shared.BasicScenarios;
    using CloudNimble.DotNetDocs.Tests.Shared.Parameters;
    ```
  </Step>
  
  <Step title="Create Your First Instance">
    Start using the library:
    
    ```csharp
    // Create a test base instance
    var testBase = new DotNetDocsTestBase();
    
    // Get assembly documentation
    var assembly = testBase.GetTestsDotSharedAssembly();
    
    // Work with sample classes
    var sample = new SampleClass { Name = "My Sample" };
    ```
  </Step>
</Steps>

<Accordion title="Core Classes">

  - **MixedAccessClass** - Access modifier examples
  - **Edge cases** - Special documentation scenarios
</Accordion>

## üéØ Common Use Cases

<Tabs>
  <Tab title="Basic Usage">
    ```csharp
    public class MyApp
    {
        public void BasicExample()
        {
            var sample = new SampleClass 
            { 
                Name = "Example" 
            };
            
            Console.WriteLine($"Sample: {sample.Name}");
        }
    }
    ```
  </Tab>
  
  <Tab title="Inheritance">
    ```csharp
    public class MyDerivedClass : BaseClass
    {
        public override void VirtualMethod()
        {
            base.VirtualMethod();
            Console.WriteLine("Extended behavior");
        }
    }
    ```
  </Tab>
  
  <Tab title="Disposable Pattern">
    ```csharp
    public void WorkWithResources()
    {
        using (var resource = new DisposableClass())
        {
            resource.DoWork();
            // Automatically disposed
        }
    }
    ```
  </Tab>
</Tabs>
