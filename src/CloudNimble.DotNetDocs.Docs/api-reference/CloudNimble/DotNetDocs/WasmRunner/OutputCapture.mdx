## Summary

Captures console output by redirecting [Out](https://learn.microsoft.com/dotnet/api/system.console.out) to a [StringWriter](https://learn.microsoft.com/dotnet/api/system.io.stringwriter).

## Remarks

This class is used to intercept `Console.WriteLine` calls during code execution
            so their output can be returned to the caller. Call [OutputCapture.StartCapture](/api-reference/CloudNimble/DotNetDocs/WasmRunner/OutputCapture#startcapture) before
            executing user code and [OutputCapture.StopCapture](/api-reference/CloudNimble/DotNetDocs/WasmRunner/OutputCapture#stopcapture) afterward to retrieve the output.
