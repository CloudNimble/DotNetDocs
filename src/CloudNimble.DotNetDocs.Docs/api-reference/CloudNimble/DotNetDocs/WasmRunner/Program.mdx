## Summary

Entry point for the .NET WASM runner application.

## Remarks

Exposes C# compilation and execution capabilities to JavaScript via `[JSExport]` interop.
            The exported methods are called from the `dotnet-runner.jsx` Mintlify React component.
            On runtimes that do not support `[JSExport]` (e.g., desktop .NET), the methods are still
            available for direct invocation.
