## Summary

Compiles and executes C# code using the Roslyn compiler.

## Remarks

This class accepts top-level C# statements (or full programs), compiles them in-memory
            using Roslyn, performs security validation via [SecuritySyntaxWalker](/api-reference/CloudNimble/DotNetDocs/WasmRunner/SecuritySyntaxWalker),
            captures console output via [OutputCapture](/api-reference/CloudNimble/DotNetDocs/WasmRunner/OutputCapture), and returns structured
            [CompilationResult](/api-reference/CloudNimble/DotNetDocs/WasmRunner/CompilationResult) objects.
